# Image Analyzer — AGENTS.md

## 프로젝트 개요
이미지 파일에서 Point/Rect ROI를 선택해 RGB 평균값을 분석하는 도구.
영상 파일은 지원하지 않는다. 이미지 파일만 처리한다.

## 기술 스택 (버전 고정)
- Python 3.11+
- PyQt6 6.7.x
- opencv-python 4.10.x
- numpy 1.26.x

## 기능 명세

### 파일 열기
- 지원 포맷: jpg, jpeg, png, bmp, tiff
- 메뉴바 또는 버튼으로 파일 열기

### 캔버스
- 이미지를 창 크기에 맞게 스케일링해서 표시
- 마우스 휠로 줌 인/아웃
- 줌 상태에서도 실제 픽셀 좌표 정확히 계산

### ROI 선택 (모드 전환 버튼으로 구분)
1. **Point 모드**: 클릭한 픽셀의 R, G, B 값 표시
2. **Rect 모드**: 드래그로 사각형 선택 후 영역 내 R, G, B 평균값 표시

### ROI 표시
- 여러 ROI를 동시에 화면에 유지
- 각 ROI마다 고유 색상 + 번호 라벨
- ROI 목록을 우측 패널에 테이블로 표시 (번호, 타입, R, G, B)
- 개별 ROI 삭제 버튼
- 전체 ROI 초기화 버튼

### 결과 저장
- ROI 목록을 CSV로 저장
- 현재 화면(ROI 오버레이 포함)을 PNG로 저장

3. **C++ 친화적 주석**: 파이썬/PyQt를 작성할 때 C++ 개념(포인터, GC, RAII, 가상 함수 오버라이딩, Lambda 캡처 등)에 빗대어 원리를 비교하는 주석을 반드시 담을 것.
4. **단일 문서 기반 관리**: 향후 추가되는 모든 기능 명세는 전부 본 `AGENTS.md` 하단에 기록하고, `PROGRESS.md`로 진척도를 영구히 추적할 것.

## 추가 기능 명세 (v1.1)
1. **우측 패널 제거 및 캔버스 확장**: 기존 우측의 그리드(ROI 테이블)를 완전히 제거하고 캔버스가 화면을 넓게 차지하도록 변경.
2. **직관적인 그래픽 렌더링**: 테이블 대신 캔버스 위에 그려진 ROI(Point/Rect) 바로 옆에 `R: 값, G: 값, B: 값` 텍스트 레이블을 직접 겹쳐서(Overlay) 이쁘게 그려줄 것.
3. **결과 저장 버튼 교체**: 쓸모없어진 'CSV 저장' 버튼은 제거하고, 'PNG(결과캡처) 저장' 버튼은 상단 중앙 툴바로 이동시킬 것.
4. **우클릭 삭제 기능**: 테이블의 개별 삭제 버튼이 사라지므로, 캔버스에 찍힌 특정 ROI를 마우스 **우클릭**하면 해당 ROI가 삭제되는 기능을 구현할 것.

## 추가 기능 명세 (v1.2)
1. **캔버스 배경색 구분**: 이미지가 출력되지 않는 빈 캔버스 여백 부분(Viewport)을 눈에 띄지 않는 차분한 어두운 톤(예: Dark Gray)으로 칠해서, 실제 이미지 영역과 시각적으로 깔끔하게 분리합니다.
2. **화면 맞춤(Fit Window) 버튼**: 줌/팬(이동)을 하다가 뷰가 엉켰을 때, 한 번의 클릭으로 이미지를 다시 캔버스 정중앙에 딱 맞게 스케일링해 주는 '화면 맞춤' 버튼을 툴바에 추가합니다.
3. **우측 영상/히스토그램 패널 부활**: Rect(영역) 모드로 ROI를 생성했을 때, 우측에 전용 패널을 살려 **선택한 영역만 잘라낸 작고 예쁜 도무송 이미지(Crop View)**와, 해당 영역 내 픽셀들의 **RGB 히스토그램 그래프**를 그려 보여줍니다. 플롯(그래프) 처리는 별도의 무거운 외부 패키지 설치 없이 `cv2.calcHist`를 응용하여 가볍고 빠르게 그리도록 설계합니다.

## 추가 기능 명세 (v1.3)
1. **전체 다크 테마 적용**: 하얀색 바탕이나 검은색 글씨 등 운영체제 기본 GUI 테마 스타일을 버리고, 전체를 Visual Studio 코딩 환경 느낌의 다크 그레이 톤으로 세팅합니다. 버튼 가시성 확보를 위해 글자는 하얗게, 호버(Hover) 시 효과를 부여합니다.
2. **패널 분리선 디자인**: 배경색이 하얗게 둥둥 떠서 어색하던 각 패널(메뉴바, 캔버스, 우측 메뉴) 사이의 간격을 1~2 픽셀의 VS 스타일 파란색 테두리(`Thin Blue Line`)로 구분하여 전문 사진 분석 도구의 느낌을 줍니다.

## 추가 기능 명세 (v1.4)
1. **히스토그램 수치화(Grids & Axis)**: 그래프 이미지 테두리에 X축(0, 127, 255) 구간과 Y축(Max 픽셀수) 눈금을 텍스트 및 가이드라인(Grid)으로 그려 수치 시계성을 확보합니다.
2. **상세 통계 지표 패널**: 히스토그램 하단 공간에 R, G, B 채널별 평균(Mean), 표준편차(StdDev), 최대/최소값 등 구체적이고 전문적인 통계 분석치를 직접 숫자로 확인할 수 있는 전용 텍스트 컨테이너 라벨을 신설합니다.
